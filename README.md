# iclicker_monitor

IClicker Monitor 是一个用于监测屏幕并自动执行点击操作的工具，主要用于辅助在线课堂互动。

## 功能特点

### 1. 课程管理
- 手动添加、修改、删除课程信息
- 每节课包含：星期几、起止时间、课号和课程名称
- **课程图标上传功能**：支持上传课程在iclicker网站上的官方图标截图
- 数据以JSON文件和CSV文件形式保存在软件安装目录
- 下次打开软件自动加载课程数据和图标信息

### 2. 屏幕监测（开发中）
- 识别屏幕上的文字和图案
- 监测指定课程的互动界面

### 3. 自动操作（开发中）
- 在识别到相应内容后自动点击
- 模拟人类操作，避免被检测为自动化工具

## 安装说明

1. 确保您的系统已安装Python 3.6或更高版本
2. 将本项目的所有文件下载到本地
3. 安装必要的依赖（本项目主要使用Python内置库）


## 每学期换课指南

当您在每学期更换课程后，需要按照以下步骤更新系统配置：

1. **更新课程表**：
   - 运行‘课程管理系统’
   - 删除不再需要的旧课程
   - 添加新课程信息：
         - 星期几
         - 起止时间 （24小时制）
         - 课号和课程名称填一样的
   - 确保课程名称和课程代码的准确性

  示例：

2. **更新课程截图**：
   - 登录iClicker网站
   - 截取每门新课程在iClicker网站上的官方图标
   - 点击"选择截图"按钮上传课程图标
   - 确保截图文件名与课程名称完全一致（包括大小写和特殊字符）
   - 上传的截图会自动保存到 `img/course` 文件夹

请主动检查img/course路径下的截图是否存在，如果没有则需要手动保存截图并命名

示例：

3. **重要注意事项**：
   - **课程名称匹配**：更新课程表时，课程名称和课程代码必须与图片文件名称完全一致，否则系统无法正确识别课程
   - **屏幕设置**：使用软件时，请确保电脑屏幕不会自动关闭或进入睡眠模式，否则可能影响屏幕监测和自动点击功能
   - **保存更改**：完成所有更新后，务必点击"保存课程表"按钮保存所有更改

## 主程序使用方法

1. 运行自动答题系统：
2. 打开iclicker 并使网页全屏
3. 确保自动答题系统弹窗正在运行，且iclicker网页并无任何遮挡
4. 点击‘鼠标控制开始’
