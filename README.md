# iclicker_monitor

IClicker Monitor 是一个用于监测屏幕并自动执行点击操作的工具，主要用于辅助在线课堂互动。

## 功能特点

### 1. 课程管理
- 手动添加、修改、删除课程信息
- 每节课包含：星期几、起止时间、课号和课程名称
- 数据以JSON文件形式保存在软件安装目录
- 下次打开软件自动加载课程数据

### 2. 屏幕监测（开发中）
- 识别屏幕上的文字和图案
- 监测指定课程的互动界面

### 3. 自动操作（开发中）
- 在识别到相应内容后自动点击
- 模拟人类操作，避免被检测为自动化工具

## 安装说明

2. 将本项目的所有文件下载到本地
