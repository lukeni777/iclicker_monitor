# iclicker_monitor

IClicker Monitor 是一个用于监测屏幕并自动执行点击操作的工具，主要用于辅助在线课堂互动。

## 功能特点

### 1. 课程管理
- 手动添加、修改、删除课程信息
- 每节课包含：星期几、起止时间、课号和课程名称
- **课程图标上传功能**：支持上传课程在iclicker网站上的官方图标截图
- 数据以JSON文件和CSV文件形式保存在软件安装目录
- 下次打开软件自动加载课程数据和图标信息

### 2. 屏幕监测（开发中）
- 识别屏幕上的文字和图案
- 监测指定课程的互动界面

### 3. 自动操作（开发中）
- 在识别到相应内容后自动点击
- 模拟人类操作，避免被检测为自动化工具

## 安装说明

1. 确保您的系统已安装Python 3.6或更高版本
2. 将本项目的所有文件下载到本地
3. 安装必要的依赖（本项目主要使用Python内置库）

## 使用方法

### 课程管理使用指南

1. 运行课程管理程序：
   ```
   python course_gui.py
   ```

2. 添加课程：
   - 填写课程信息（星期几、起止时间、课号、课程名称）
   - 点击"选择截图"按钮上传课程图标
   - 选择iclicker网站上的课程官方图标截图
   - 点击"添加课程"按钮保存课程

3. 课程图标处理：
   - 上传的截图会自动保存到 `img/course` 文件夹
   - 文件名使用课程名称（自动移除特殊字符）
   - 保留原截图的文件扩展名
   - 相同课程的新截图会自动替换旧截图

4. 管理课程：
   - 从课程列表中选择课程进行编辑或删除
   - 点击"保存课程表"按钮将数据保存到文件
